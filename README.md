# 🚗 German Cars Data Analysis (Python)

## 📊 Overview
Analyzed 46,000+ German car listings from AutoScout24 (2011–2021) to uncover pricing trends, fuel preferences, and performance insights.

## 📁 Dataset
- 46,405 rows, 9 features
- Features: price, mileage, fuel type, transmission, horsepower, year

## 🔍 Key Insights
- Price has strong positive correlation with horsepower (~0.75)
- 90% of cars have <200 HP
- 76% of cars are priced below €20,000
- Automatic cars increased after 2017 and overtook manual cars
- Electric & hybrid cars increased with time
- Diesel peaked in 2017 (data anomaly)

## 📈 Analysis Performed
- Data Cleaning (null handling, formatting)
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Trend Analysis (year, fuel, transmission)
- Visualization using Matplotlib
